# VirtualFit — AI Virtual Try-On

An AI-powered virtual try-on app for online clothing stores. Upload your photo, pick a garment from the catalog, and see yourself wearing it — generated by Google Gemini.

## Features

- **Virtual try-on** — Upload a photo and get a photorealistic image of yourself wearing any garment from the catalog
- **Product catalog** — Browse tops, pants, outerwear, and swimwear with filters and product detail pages
- **Combinar / Cambiar** — From the result view, either replace the current garment or layer a new one on top of the generated look
- **Garment extraction** — The AI identifies and applies only the named product from a photo, ignoring any other clothing worn by the model in the product image
- **Fit context** — Enter your size (XS–2XL or numeric), height, weight, and age to inform the AI's proportions and draping
- **Marea swimwear** — 25 one-piece swimsuits from the Marea collection
- **LAN server** — Run on your local network and test from your phone

## Tech stack

- [Next.js 15](https://nextjs.org) App Router
- [tRPC v11](https://trpc.io)
- [Tailwind CSS v4](https://tailwindcss.com)
- [TypeScript](https://www.typescriptlang.org)
- [Google Generative AI](https://ai.google.dev) — `gemini-2.5-flash` for image descriptions + `gemini-3.1-flash-image` for generation

## How it works

1. **Step 1 — Describe**: Two parallel calls to `gemini-2.5-flash` produce detailed text descriptions of the person and the specific garment (ignoring any outfit on the product model)
2. **Step 2 — Generate**: `gemini-3.1-flash-image` receives both descriptions plus optional body measurements and renders a photorealistic try-on image
3. **Layering**: When combining garments, the previously generated image becomes the base for the next generation, letting you stack a shirt, pants, jacket, etc.

## Getting started

1. Clone the repo and install dependencies:

```bash
npm install
```

2. Copy `.env.example` to `.env` and add your Google AI key:

```env
GOOGLE_AI_KEY="your-key-here"
```

Get a free API key at [aistudio.google.com](https://aistudio.google.com).

3. Start the dev server:

```bash
npm run dev
```

To expose it on your local network (for testing on mobile):

```bash
npx next dev -H 0.0.0.0 --port 3000
```

Then open `http://<your-local-ip>:3000` on your phone.

```
  garments/               # Product images

```
